 Jogo Ponte de Vidro
Um jogo interativo de sorte e estratégia inspirado no desafio das pontes de vidro. O objetivo é atravessar três seções sem escolher o vidro frágil que se quebra.

 Como Jogar
Início: Na tela inicial, clique em "INICIAR O JOGO!".

O Desafio: Você verá três pontes consecutivas. Em cada uma, você deve escolher um número de 1 a 3.

A Sorte: Um dos números representa o "vidro frágil" gerado aleatoriamente pelo sistema.

Se você escolher um número diferente do vidro frágil, você avança para a próxima ponte.

Se você escolher o número igual ao vidro frágil, você cai e o jogo termina.

Vitória: Atravesse as 3 pontes com sucesso para vencer o desafio!

 Funcionalidades
Lógica Aleatória: O "vidro frágil" é sorteado dinamicamente a cada rodada usando Math.random().

Feedback Visual:

Efeito de Tremor (Shake): A tela vibra e fica vermelha quando o jogador erra.

Efeito de Vitória: Um brilho dourado e animações de pulso celebram a vitória.

Interface Responsiva: O layout se adapta automaticamente a telas de celulares, tablets e computadores.

Interatividade por Teclado: É possível confirmar sua escolha pressionando a tecla Enter.

Prevenção de Erros: O sistema valida se o número digitado está entre 1 e 3.

 Tecnologias Utilizadas
O projeto foi construído puramente com tecnologias web padrão (Vanilla Stack):

HTML5: Estruturação das telas e elementos de entrada.

CSS3: Estilização moderna com Flexbox, Glassmorphism (transparências) e @keyframes para animações.

JavaScript: Controle de estado do jogo, manipulação de eventos e lógica de vitória/derrota.

Faça o download dos arquivos ou clone o repositório.

Certifique-se de que a estrutura de pastas (especialmente a pasta imagem) esteja correta.

Abra o arquivo index.html em qualquer navegador moderno.
